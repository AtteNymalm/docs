= Confirm Dialog

// tag::description[]
Confirm Dialog is an easy to use component to ask the user to confirm a choice.
// end::description[]
It supports showing OK/Cancel or Yes/No/Cancel style dialog to confirm a potentially dangerous operation or an irreversible action. Additionally you can show alert style dialog for informing user about events that require immediate user action. You can customize the dialog content and buttons.
